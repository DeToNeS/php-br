Projeto PHP-BR
========================

Este projeto tem como objetivo o desenvolvimento
de um portal para a comunidade php no Brasil.

```
irc.freenode.net
##php-br
```

1) Instalacao
----------------------------------

    curl -s http://getcomposer.org/installer | php

    php composer.phar install
